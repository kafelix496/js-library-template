# js-library-template
Javascript library template

## Need to change custom config names and stuff
> Recommend to use initialize scripts
``` node scripts/initConfig.js ```

## Lint
```
npm run lint
```

## Unit Test
```
npm run test
```

## Build
```
npm run build
```

## Examples
```
es : npm start 
cjs : node index.js
umd : npx serve
```
